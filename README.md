## Course Overview

This course introduces the fundamental concepts of Object-Oriented Programming (OOP) using Java. Students will learn to design and implement programs using classes, objects, inheritance, polymorphism, encapsulation, and abstraction.

## How to Use This Repository

### Clone the Repository
```bash
git clone https://github.com/massoudhamad/pt821.git
cd pt821
```

### Compile and Run Java Programs
```bash
javac examples/HelloWorld.java
java -cp examples HelloWorld
```

### Using BlueJ
1. Open BlueJ
2. Select **Project > Open Non BlueJ...**
3. Navigate to the cloned folder
4. Open any `.java` file to explore and run
